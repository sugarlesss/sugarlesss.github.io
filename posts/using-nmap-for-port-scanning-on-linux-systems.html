Linux 使用 nmap 进行端口扫描

 2019-05-19  约 221 字   预计阅读 1 分钟 
目录
  • 一、安装 nmap

目前端口扫描可以通过 web 版本的端口扫描进行简单的测试，一次性扫描的端口数量一般也会有限制。通常情况下，如果仅仅是扫描一些常用端口，如上图所示的网页版端口扫描可以满足。如果想折腾一下，或者是对自己的主机进行安全检查之类的，还是使用 Linux 系统下的 nmap 工具进行扫描会比较方便一点。

一、安装 nmap

使用 yum 安装 nmap

#更新一下源
yum update -y

#安装
yum install nmap -y

二、nmap 命令

#扫描本机开放的端口「127.0.0.1 一般指代本机」
nmap 127.0.0.1  -p 1-65535
